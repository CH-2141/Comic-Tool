# Comic-Tool

**Comic-Tool（漫展战术终端）** 是一个为漫展参展者设计的摊位管理与购买辅助工具，用于在漫展现场高效记录、筛选和管理摊位信息，避免漏买、重复逛摊或现场决策混乱的问题。

本项目以纯前端静态网页形式实现，可直接通过 GitHub Pages 使用，无需安装、无需后端服务。

---

## ✨ 功能特性

- 📋 **摊位信息管理**
  - 查看摊位名称、商品信息与备注
  - 快速浏览与定位目标摊位

- 🔍 **筛选与排序**
  - 根据状态、优先级等条件筛选摊位
  - 辅助制定合理的逛展路线

- ✅ **状态标记**
  - 已购买（GET）
  - 已售罄（SOLD OUT）
  - 自定义必买指数 / 重要程度

- 🗺️ **漫展地图支持**
  - 导入漫展场地图
  - 在地图上标记摊位位置
  - 支持缩放、拖动与交互查看

- 💾 **本地数据保存**
  - 数据保存在浏览器本地
  - 刷新页面不会丢失记录
  - 无需登录，不上传任何个人数据

- 🔁 **数据导入与备份**
  - 支持数据导出与恢复
  - 便于多设备或多场漫展使用

## 🚀 使用方法

1. 打开在线页面或本地页面
2. 浏览摊位列表并点击查看详情
3. 标记摊位状态（已买 / 售罄 / 优先级）
4. 使用地图视图规划现场路线
5. 在工具菜单中进行数据备份或导入

## 🛠 技术说明

- 纯前端实现（HTML / CSS / JavaScript）
- 无后端、无数据库依赖
- 基于浏览器 `LocalStorage` 进行数据持久化
- 支持离线使用，适合漫展现场网络不稳定环境
- 可直接部署至 GitHub Pages 或任意静态托管平台
